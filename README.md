# RSS reader

Simple RSS reader. Project for Hexlet course.

## Demo

https://frontend-project-lvl3-someden.vercel.app

## Setup

```sh
make install
```

## Run

```sh
make start
```

## Lint

```sh
make lint
```

## Build

```sh
make build
```
